## 값(value), 식(expression) 문(statment)

1. 값 : 데이터 그 자체
2. 식 : 하나의 값으로 평가될 수 있는 코드 조각
3. 문 : 프로그램에서 특정 작업을 수행

- 식은 값을 반환한다.
- 문은 특정 작업을 수행한다.
- 식은 문이 될 수 있다.

```javascript
// 5 + 3 은 식
let result = 5 + 3;
```

- 문은 식이 될 수 없다.

```javascript
//if문은 식이 될 수 없음
let x = if (y > 10) { 20 } else { 30 };
```
